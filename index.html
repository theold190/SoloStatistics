    • Статистика онлайн-курса "Соло на клавиатуре" в твоем браузере

    Расширение для браузера Google Chrome для отображения прогресса в онлайн-курсах "Соло на клавиатуре" (ergosolo.ru)

    Project maintained by theold190 Hosted on GitHub Pages — Theme by mattgraham

    ErgoSoloStatistics позволяет удобно отслеживать свой прогресс в онлайн-курсе "Соло на клавиатуре". "Соло на клавиатуре" - авторский курс разработанный Владимиром Шахиджаняном, психолога, журналиста, преподавателя. Подробнее о курсе можно узнать по адресу http://nabiraem.ru/about/.

    Расширение работает только для браузера Google Chrome. Для установки расширения необходимо открыть файл по этой ссылке и пройти стандартную процедуру установки.

    Расширение позволяет отображать свой текущий прогресс буквально за одно нажатие кнопки мыши. Вот примерно как выглядит текущий статус:



    С помощью этого расширения легко отслеживать как свой прогресс, так и прогрес любого другого ученика. Для этого в разделе настроек необходимо внести идентификационный номер ученика с сайта nabiraem.ru и сохранить изменения. Сам раздел настроек выглядит следующим образом:



    Как получить идентификационный номер ученика с сайта nabiraem.ru?

    1. Зайти под учетной записью ученика на сайт nabiraem.ru
    2. Найти себя в разделе "Солисты в онлайн"
    3. Щелкнуть правой кнопкой мышки и выбрать "Скопировать ссылку"
    4. Вставить ссылку в любой текстовый редактор.
    5. Получится текст похожий на http://nabiraem.ru/user/NNNNN, где NNNNN и есть искомый номер ученика.
